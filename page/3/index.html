Spring Security14 - еҹәдәҺй…ҚзҪ®зҡ„жҺҘеҸЈи§’иүІи®ҝй—®жҺ§еҲ¶

Posted on 2021-07-12 In Spring Security

1. еүҚиЁҖ

еҜ№дәҺеҸ—йҷҗзҡ„и®ҝй—®иө„жәҗпјҢе№¶дёҚжҳҜеҜ№жүҖжңүи®ӨиҜҒйҖҡиҝҮзҡ„з”ЁжҲ·ејҖж”ҫзҡ„гҖӮжҜ”еҰӮ A з”ЁжҲ·зҡ„и§’иүІжҳҜдјҡи®ЎпјҢйӮЈд№Ҳд»–е°ұеҸҜд»Ҙи®ҝй—®иҙўеҠЎзӣёе…ізҡ„иө„жәҗгҖӮB з”ЁжҲ·жҳҜдәәдәӢпјҢйӮЈд№Ҳд»–еҸӘиғҪи®ҝй—®дәәдәӢзӣёе…ізҡ„иө„жәҗгҖӮжҲ‘д»¬еңЁдёҠдёҖж–Үдёӯд№ҹеҜ№еҹәдәҺи§’иүІзҡ„и®ҝй—®жҺ§еҲ¶зҡ„зӣёе…іжҰӮеҝөиҝӣиЎҢдәҶжҺўи®ЁгҖӮеңЁе®һйҷ…ејҖеҸ‘дёӯжҲ‘д»¬еҰӮдҪ•еҜ№иө„жәҗиҝӣиЎҢи§’иүІзІ’еәҰзҡ„з®ЎжҺ§е‘ўпјҹд»ҠеӨ©жҲ‘жқҘе‘ҠиҜүдҪ  Spring Security жҳҜеҰӮдҪ•жқҘи§ЈеҶіиҝҷдёӘй—®йўҳзҡ„гҖӮ

2. е°Ҷи§’иүІеҶҷе…Ҙ UserDetails

жҲ‘д»¬дҪҝз”Ё UserDetailsService еҠ иҪҪ UserDetails ж—¶д№ҹдјҡжҠҠз”ЁжҲ·зҡ„ GrantedAuthority жқғйҷҗйӣҶеҶҷе…Ҙе…¶дёӯгҖӮдҪ еҸҜд»Ҙе°Ҷи§’иүІжҢҒд№…еҢ–е№¶еңЁиҝҷдёӘзӮ№иҝӣиЎҢжіЁе…Ҙз„¶еҗҺй…ҚзҪ®и®ҝй—®зӯ–з•ҘпјҢеҗҺз»ӯзҡ„й—®йўҳдәӨз»ҷ Spring Security гҖӮ

3. еңЁ HttpSecurity дёӯиҝӣиЎҢй…ҚзҪ®и§’иүІи®ҝй—®жҺ§еҲ¶

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮй…ҚзҪ® WebSecurityConfigurerAdapter дёӯзҡ„ HttpSecurity жқҘжҺ§еҲ¶жҺҘеҸЈзҡ„и§’иүІи®ҝй—®гҖӮ

3.1 йҖҡиҝҮеҲӨж–ӯз”ЁжҲ·жҳҜеҗҰжҢҒжңүи§’иүІжқҘиҝӣиЎҢи®ҝй—®жҺ§еҲ¶

иЎЁзӨәжҢҒжңү ROLE_ADMIN и§’иүІзҡ„з”ЁжҲ·жүҚиғҪи®ҝй—® /foo/test жҺҘеҸЈгҖӮжіЁж„ҸпјҡhasRole(String role) ж–№жі•е…ҘеҸӮдёҚиғҪжҗәеёҰеүҚзјҖ ROLE_ гҖӮжҲ‘д»¬жқҘжҹҘзңӢ SecurityExpressionRoot дёӯзӣёе…іжәҗз Ғ:

еҫҲжҳҺжҳҫ hasRole ж–№жі•жәҗдәҺ hasAnyRole (жҢҒжңүд»»дҪ•е…¶дёӯи§’иүІд№ӢдёҖпјҢе°ұиғҪж»Ўи¶іи®ҝй—®жқЎд»¶пјҢз”ЁдәҺдёҖдёӘжҺҘеҸЈејҖж”ҫз»ҷеӨҡдёӘи§’иүІи®ҝй—®ж—¶) :

еҰӮжһңдёҖдёӘжҺҘеҸЈејҖж”ҫз»ҷеӨҡдёӘи§’иүІпјҢжҜ”еҰӮ /foo/test ејҖж”ҫз»ҷдәҶ ROLE_APP е’Ң ROLE_ADMIN еҸҜд»Ҙиҝҷд№ҲеҶҷпјҡ

hasAnyRole ж–№жі•жңҖз»Ҳзҡ„е®һзҺ°дёә hasAnyAuthorityName(String prefix, String... roles):

дёҠйқўжүҚжҳҜж №жң¬зҡ„е®һзҺ°, йңҖиҰҒдёҖдёӘ prefix е’ҢжҜҸдёҖдёӘ role иҝӣиЎҢжӢјжҺҘпјҢз„¶еҗҺз”ЁжҲ·зҡ„и§’иүІйӣҶеҗҲ roleSet дёӯеҢ…еҗ«дәҶе°ұиҝ”еӣһtrueж”ҫиЎҢпјҢеҗҰеҲҷе°ұ false жӢ’з»қгҖӮй»ҳи®Өзҡ„ prefix дёә defaultRolePrefix= ROLE_гҖӮ

3.2 йҖҡиҝҮеҲӨж–ӯз”ЁжҲ·зҡ„ GrantedAuthority жқҘиҝӣиЎҢи®ҝй—®жҺ§еҲ¶

жҲ‘д»¬д№ҹеҸҜд»ҘйҖҡиҝҮ hasAuthority е’Ң hasAnyAuthority жқҘеҲӨе®ҡгҖӮ е…¶е®һеә•еұӮе®һзҺ°е’Ң hasAnyRole ж–№жі•дёҖж ·пјҢеҸӘдёҚиҝҮ prefix дёә null гҖӮд№ҹе°ұжҳҜдҪ еҶҷе…Ҙзҡ„ GrantedAuthority жҳҜд»Җд№Ҳж ·еӯҗзҡ„пјҢиҝҷйҮҢдј е…ҘеҸӮж•°зҡ„е°ұжҳҜд»Җд№Ҳж ·еӯҗзҡ„пјҢдёҚеҶҚеҸ— ROLE_ еүҚзјҖзҡ„еҲ¶зәҰгҖӮ

4. еҢҝеҗҚи®ҝй—®

еҢҝеҗҚиә«д»ҪйӘҢиҜҒзҡ„з”ЁжҲ·е’ҢжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„з”ЁжҲ·д№Ӣй—ҙжІЎжңүзңҹжӯЈзҡ„жҰӮеҝөе·®ејӮгҖӮSpring Security зҡ„еҢҝеҗҚиә«д»ҪйӘҢиҜҒеҸӘжҳҜдёәжӮЁжҸҗдҫӣдәҶдёҖз§Қжӣҙж–№дҫҝзҡ„ж–№ејҸжқҘй…ҚзҪ®и®ҝй—®жҺ§еҲ¶еұһжҖ§гҖӮжүҖжңүзҡ„еҢҝеҗҚз”ЁжҲ·йғҪжҢҒжңүи§’иүІ ROLE_ANONYMOUS гҖӮжүҖд»ҘдҪ еҸҜд»ҘдҪҝз”Ё 2.1 е’Ң 2.2 з« иҠӮзҡ„ж–№жі•жқҘй…ҚзҪ®еҢҝеҗҚи®ҝй—®:

дҪ д№ҹеҸҜд»ҘйҖҡиҝҮд»ҘдёӢж–№ејҸиҝӣиЎҢй…ҚзҪ®пјҡ

6. permitAll дёҺ anonymous зҡ„дёҖдәӣжҺўи®Ё

ејҖж”ҫиҜ·жұӮ е…¶е®һйҖҡеёёжғ…еҶөдёӢи·ҹ еҢҝеҗҚиҜ·жұӮ жңүдәӨеҸүгҖӮе®ғд»¬зҡ„дё»иҰҒеҢәеҲ«еңЁдәҺпјҡ еҪ“еүҚзҡ„ Authentication дёә null ж—¶ permitAll жҳҜж”ҫиЎҢзҡ„пјҢиҖҢ anonymous йңҖиҰҒ Authentication дёә AnonymousAuthenticationToken гҖӮиҝҷйҮҢжҳҜжҜ”иҫғйҡҫд»ҘзҗҶи§Јзҡ„пјҢдёӢйқўжҳҜжқҘиҮӘ Spring ж–ҮжЎЈдёӯзҡ„дёҖдәӣдҝЎжҒҜпјҡ

йҖҡеёёпјҢйҮҮз”ЁвҖңй»ҳи®ӨжӢ’з»қвҖқзҡ„еҒҡжі•иў«и®ӨдёәжҳҜдёҖз§ҚиүҜеҘҪзҡ„е®үе…ЁеҒҡжі•пјҢеңЁиҜҘж–№жі•дёӯпјҢжӮЁжҳҺзЎ®жҢҮе®ҡе…Ғи®ёзҡ„еҶ…е®№пјҢе№¶зҰҒжӯўе…¶д»–жүҖжңүеҶ…е®№гҖӮе®ҡд№үжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„з”ЁжҲ·еҸҜд»Ҙи®ҝй—®зҡ„еҶ…е®№зҡ„жғ…еҶөдёҺжӯӨзұ»дјјпјҢе°Өе…¶жҳҜеҜ№дәҺWebеә”з”ЁзЁӢеәҸгҖӮи®ёеӨҡз«ҷзӮ№иҰҒжұӮз”ЁжҲ·еҝ…йЎ»йҖҡиҝҮиә«д»ҪйӘҢиҜҒжүҚиғҪдҪҝз”Ёе°‘ж•°еҮ дёӘURLпјҲдҫӢеҰӮпјҢдё»йЎөе’Ңзҷ»еҪ•йЎөйқўпјүгҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢжңҖз®ҖеҚ•зҡ„жҳҜдёәиҝҷдәӣзү№е®ҡзҡ„URLе®ҡд№үи®ҝй—®й…ҚзҪ®еұһжҖ§пјҢиҖҢдёҚжҳҜдёәжҜҸдёӘеҸ—дҝқжҠӨзҡ„иө„жәҗе®ҡд№үи®ҝй—®й…ҚзҪ®еұһжҖ§гҖӮжҚўеҸҘиҜқиҜҙпјҢжңүж—¶еҫҲй«ҳе…ҙең°иҜҙй»ҳи®Өжғ…еҶөдёӢйңҖиҰҒROLE_SOMETHINGпјҢе№¶дё”еҸӘе…Ғи®ёиҜҘи§„еҲҷзҡ„жҹҗдәӣдҫӢеӨ–пјҢдҫӢеҰӮеә”з”ЁзЁӢеәҸзҡ„зҷ»еҪ•пјҢжіЁй”Җе’Ңдё»йЎөгҖӮжӮЁиҝҳеҸҜд»Ҙд»ҺиҝҮж»ӨеҷЁй“ҫдёӯе®Ңе…ЁеҝҪз•ҘиҝҷдәӣйЎөйқўпјҢд»ҺиҖҢз»•иҝҮи®ҝй—®жҺ§еҲ¶жЈҖжҹҘпјҢ
иҝҷе°ұжҳҜжҲ‘д»¬жүҖиҜҙзҡ„еҢҝеҗҚиә«д»ҪйӘҢиҜҒгҖӮ

дҪҝз”Ё permitAll() е°Ҷй…ҚзҪ®жҺҲжқғпјҢд»ҘдҫҝеңЁиҜҘзү№е®ҡи·Ҝеҫ„дёҠе…Ғи®ёжүҖжңүиҜ·жұӮпјҲжқҘиҮӘеҢҝеҗҚз”ЁжҲ·е’Ңе·Ізҷ»еҪ•з”ЁжҲ·пјү,anonymous() дё»иҰҒжҳҜжҢҮз”ЁжҲ·зҡ„зҠ¶жҖҒпјҲжҳҜеҗҰзҷ»еҪ•пјүгҖӮеҹәжң¬дёҠпјҢзӣҙеҲ°з”ЁжҲ·иў«вҖңи®ӨиҜҒвҖқдёәжӯўпјҢе®ғе°ұжҳҜвҖңеҢҝеҗҚз”ЁжҲ·вҖқгҖӮе°ұеғҸжҜҸдёӘдәәйғҪжңүвҖңй»ҳи®Өи§’иүІвҖқдёҖж ·гҖӮ

Spring Security13 - RBACжқғйҷҗжҺ§еҲ¶жҰӮеҝөзҡ„зҗҶи§Ј

Posted on 2021-07-12 In Spring Security

1. еүҚиЁҖ

жҲӘжӯўеҲ°дёҠдёҖзҜҮжҲ‘д»¬е·Із»ҸиғҪеӨҹз®ҖеҚ•еҒҡеҲ°з”ЁжҲ·дё»дҪ“и®ӨиҜҒеҲ°жҺҘеҸЈзҡ„и®ҝй—®жҺ§еҲ¶дәҶпјҢдҪҶжҳҜдҫқз„¶ж»Ўи¶ідёҚдәҶе®һйҷ…з”ҹдә§зҡ„йңҖиҰҒгҖӮ еҰӮжһңжҲ‘д»¬йңҖиҰҒдёҖдёӘе®Ңж•ҙзҡ„жқғйҷҗз®ЎзҗҶзі»з»ҹе°ұеҝ…йЎ»дәҶи§ЈдёҖдёӢRBAC(Role-Based Access ControlеҹәдәҺи§’иүІзҡ„и®ҝй—®жҺ§еҲ¶) зҡ„жқғйҷҗжҺ§еҲ¶жЁЎеһӢгҖӮ

2. дёәд»Җд№ҲйңҖиҰҒRBACпјҹ

еңЁжӯЈејҸи®Ёи®әRBACжЁЎеһӢд№ӢеүҚпјҢжҲ‘д»¬иҰҒжҖқиҖғдёҖдёӘй—®йўҳпјҢдёәд»Җд№ҲжҲ‘д»¬иҰҒеҒҡи§’иүІжқғйҷҗзі»з»ҹпјҹ зӯ”жЎҲеҫҲжҳҺжҳҫпјҢдёҖдёӘзі»з»ҹиӮҜе®ҡе…·жңүдёҚеҗҢи®ҝй—®жқғйҷҗзҡ„з”ЁжҲ·гҖӮжҜ”еҰӮд»ҳиҙ№з”ЁжҲ·е’Ңйқһд»ҳиҙ№з”ЁжҲ·зҡ„жқғйҷҗпјҢеҰӮжһңдҪ жҳҜ QQйҹід№җзҡ„дјҡе‘ҳйӮЈд№ҲдҪ иғҪеҗ¬й«ҳйҹіиҙЁзҡ„жӯҢжӣІпјҢеҰӮжһңдёҚжҳҜе°ұдёҚиғҪдә«еҸ—жҹҗдәӣдҫҝеҲ©зҡ„гҖҒдјҳиҙЁзҡ„жңҚеҠЎгҖӮйӮЈд№ҲиҝҷжҳҜдёҖжҲҗдёҚеҸҳзҡ„еҗ—пјҹеҸҲж—¶еҖҷдёәдәҶжөҒйҮҸеўһй•ҝжҲ–иҖ…жӢүж–°зҡ„йңҖиҰҒпјҢжҲ‘д»¬еҸҲеҸҜиғҪжҠҠдёҖдәӣеҺҹжқҘе……й’ұжүҚиғҪдә«еҸ—зҡ„жңҚеҠЎдёӢж”ҫз»ҷе…Қиҙ№з”ЁжҲ·гҖӮеҰӮжһңдҪ жңүдәҶдјҡе‘ҳзӯүзә§йӮЈе°ұжӣҙеҠ еӨҚжқӮдәҶпјҢVIP1и·ҹVIP2е…·жңүзҡ„еҠҹиғҪиӮҜе®ҡеҸҲжңүжүҖе·®еҲ«дәҶгҖӮдё»жөҒзҡ„жқғйҷҗз®ЎзҗҶзі»з»ҹйғҪжҳҜ RBAC жЁЎеһӢзҡ„еҸҳеҪўе’Ңиҝҗз”ЁпјҢеҸӘжҳҜж №жҚ®дёҚеҗҢзҡ„дёҡеҠЎе’Ңи®ҫи®Ўж–№жЎҲпјҢе‘ҲзҺ°дёҚеҗҢзҡ„жҳҫзӨәж•ҲжһңгҖӮ

дёӢеӣҫеұ•зӨәдәҶз”ЁжҲ·е’Ңи§’иүІд»ҘеҸҠиө„жәҗзҡ„з®ҖеҚ•е…ізі»пјҡ

йӮЈдёәд»Җд№ҲдёҚзӣҙжҺҘз»ҷз”ЁжҲ·еҲҶй…ҚжқғйҷҗпјҢиҝҳеӨҡжӯӨдёҖдёҫзҡ„еўһеҠ и§’иүІиҝҷдёҖзҺҜиҠӮе‘ўпјҹеҪ“з„¶зӣҙжҺҘз»ҷз”ЁжҲ·е…·дҪ“зҡ„иө„жәҗи®ҝй—®жҺ§еҲ¶жқғйҷҗд№ҹдёҚжҳҜдёҚеҸҜд»ҘгҖӮеҸӘжҳҜиҝҷж ·еҒҡзҡ„иҜқе°ұе°‘дәҶдёҖеұӮе…ізі»пјҢжү©еұ•жҖ§ејұдәҶи®ёеӨҡгҖӮеҰӮжһңдҪ зҡ„зі»з»ҹи¶іеӨҹз®ҖеҚ•е°ұдёҚиҰҒжҠҳи…ҫRBACдәҶпјҢжҖҺд№Ҳз®ҖеҚ•е°ұжҖҺд№ҲзҺ©гҖӮеҰӮжһңдҪ зҡ„зі»з»ҹйңҖиҰҒиҖғиҷ‘жү©еұ•жҖ§е’ҢжқғйҷҗжҺ§еҲ¶зҡ„еӨҡж ·жҖ§е°ұеҝ…йЎ»иҖғиҷ‘RBACгҖӮ

еҰӮжһңдҪ жңүеӨҡдёӘе…·жңүзӣёеҗҢжқғйҷҗзҡ„з”ЁжҲ·пјҢеҶҚеҲҶй…Қжқғйҷҗзҡ„ж—¶еҖҷдҪ е°ұйңҖиҰҒйҮҚеӨҚдёәз”ЁжҲ·еҺ» Query (жҹҘиҜў) е’Ң Add (иөӢдәҲ) жқғйҷҗпјҢеҰӮжһңдҪ иҰҒдҝ®ж”№пјҢжҜ”еҰӮдёҠйқўзҡ„ VIP1 еўһеҠ дёҖдёӘеҫҲ Cool зҡ„еҠҹиғҪ,дҪ е°ұиҰҒйҒҚеҺҶ VIP1 з”ЁжҲ·иҝӣиЎҢдҝ®ж”№гҖӮжңүдәҶи§’иүІеҗҺпјҢжҲ‘д»¬еҸӘйңҖиҰҒдёәиҜҘи§’иүІеҲ¶е®ҡеҘҪжқғйҷҗеҗҺпјҢе°ҶзӣёеҗҢжқғйҷҗзҡ„з”ЁжҲ·йғҪжҢҮе®ҡдёәеҗҢдёҖдёӘи§’иүІеҚіеҸҜпјҢдҫҝдәҺжқғйҷҗз®ЎзҗҶгҖӮ

еҜ№дәҺжү№йҮҸзҡ„з”ЁжҲ·жқғйҷҗи°ғж•ҙпјҢеҸӘйңҖи°ғж•ҙиҜҘз”ЁжҲ·е…іиҒ”зҡ„и§’иүІжқғйҷҗпјҢж— йңҖйҒҚеҺҶпјҢж—ўеӨ§е№…жҸҗеҚҮжқғйҷҗи°ғж•ҙзҡ„ж•ҲзҺҮпјҢеҸҲйҷҚдҪҺдәҶжјҸи°ғжқғйҷҗзҡ„жҰӮзҺҮгҖӮиҝҷж ·з”ЁжҲ·е’Ңиө„жәҗжқғйҷҗи§ЈйҷӨдәҶиҖҰеҗҲжҖ§пјҢиҝҷе°ұжҳҜRBACжЁЎеһӢзҡ„дјҳеҠҝжүҖеңЁгҖӮ

3. RBACжЁЎеһӢзҡ„еҲҶзұ»

RBACжЁЎеһӢеҸҜд»ҘеҲҶдёәпјҡRBAC0гҖҒRBAC1гҖҒRBAC2гҖҒRBAC3 еӣӣз§ҚгҖӮе…¶дёӯ RBAC0 жҳҜеҹәзЎҖпјҢе…¶е®ғдёүз§ҚйғҪжҳҜеңЁ RBAC0 еҹәзЎҖдёҠзҡ„еҸҳз§ҚгҖӮеӨ§йғЁеҲҶжғ…еҶөдёӢпјҢдҪҝз”Ё RBAC0 жЁЎеһӢе°ұеҸҜд»Ҙж»Ўи¶іеёёи§„зҡ„жқғйҷҗз®ЎзҗҶзі»з»ҹи®ҫи®ЎдәҶгҖӮдёҚиҝҮдёҖе®ҡдёҚиҰҒжӢҳжіҘдәҺжЁЎеһӢпјҢиҰҒд»ҘдёҡеҠЎйңҖиҰҒдёәе…ҲеҜјгҖӮжҺҘдёӢжқҘз®ҖеҚ•еҜ№еӣӣз§ҚжЁЎеһӢиҝӣиЎҢз®ҖеҚ•зҡ„д»Ӣз»ҚдёҖдёӢгҖӮ

3.1 RBAC0

RBAC0жҳҜеҹәзЎҖпјҢе®ҡд№үдәҶиғҪжһ„жҲҗRBACжқғйҷҗжҺ§еҲ¶зі»з»ҹзҡ„жңҖе°Ҹзҡ„йӣҶеҗҲпјҢRBAC0з”ұеӣӣйғЁеҲҶжһ„жҲҗпјҡ

  • з”ЁжҲ·пјҲUserпјү жқғйҷҗзҡ„дҪҝз”Ёдё»дҪ“
  • и§’иүІпјҲRoleпјү еҢ…еҗ«и®ёеҸҜзҡ„йӣҶеҗҲ
  • дјҡиҜқпјҲSessionпјүз»‘е®ҡз”ЁжҲ·е’Ңи§’иүІе…ізі»жҳ е°„зҡ„дёӯй—ҙйҖҡйҒ“гҖӮиҖҢдё”з”ЁжҲ·еҝ…йЎ»йҖҡиҝҮдјҡиҜқжүҚиғҪз»ҷз”ЁжҲ·и®ҫзҪ®и§’иүІгҖӮ
  • и®ёеҸҜпјҲPemissionпјү еҜ№зү№е®ҡиө„жәҗзҡ„зү№е®ҡзҡ„и®ҝй—®и®ёеҸҜгҖӮ

3.2 RBAC1

RBAC1еңЁRBAC0зҡ„еҹәзЎҖд№ӢдёҠеј•е…ҘдәҶи§’иүІз»§жүҝзҡ„жҰӮеҝөпјҢжңүдәҶз»§жүҝйӮЈд№Ҳи§’иүІе°ұжңүдәҶдёҠдёӢзә§жҲ–иҖ…зӯүзә§е…ізі»гҖӮзҲ¶и§’иүІжӢҘжңүе…¶еӯҗи§’иүІжүҖжңүзҡ„и®ёеҸҜгҖӮйҖҡдҝ—и®Іе°ұжҳҜжқҘиҜҙпјҡ дҪ иғҪе№Ізҡ„пјҢдҪ зҡ„йўҶеҜјдёҖе®ҡиғҪе№ІпјҢеҸҚиҝҮжқҘе°ұдёҚдёҖе®ҡиғҪиЎҢгҖӮ

3.3 RBAC2

еңЁдҪ“иӮІжҜ”иөӣдёӯпјҢдҪ дёҚеҸҜиғҪж—ўжҳҜиҝҗеҠЁе‘ҳеҸҲжҳҜиЈҒеҲӨе‘ҳпјҒ

иҝҷжҳҜеҫҲжңүеҗҚзҡ„дёҖеҸҘиҜқгҖӮеҸҚеә”дәҶжҲ‘д»¬з»ҸеёёеҮәзҺ°зҡ„дёҖз§ҚиҒҢеҠЎпјҲе…¶е®һд№ҹе°ұжҳҜи§’иүІпјүеҶІзӘҒгҖӮжңүдәӣи§’иүІдә§з”ҹзҡ„еҺҶеҸІеҺҹеӣ е°ұжҳҜдёәдәҶеҲ¶зәҰеҸҰдёҖдёӘи§’иүІпјҢиЈҒеҲӨе‘ҳе°ұжҳҜдёәдәҶеҲ¶зәҰиҝҗеҠЁе‘ҳд»ҺиҖҢи®©иҝҗеҠЁе‘ҳжҢүз…§и§„иҢғеҺ»жҜ”иөӣгҖӮеҰӮжһңдёҖдёӘдәәе…јд»»иҝҷдёӨдёӘи§’иүІпјҢжҜ”иөӣеҝ…з„¶е®№жҳ“еҮәзҺ°дёҚе…¬жӯЈзҡ„жғ…еҶөд»ҺиҖҢиҝқиғҢз«һжҠҖе…¬е№іжҖ§еҮҶеҲҷгҖӮиҝҳжңүе°ұжҳҜжҲ‘д»¬жҜҸдёӘдәәеңЁдёҚеҗҢзҡ„еңәжҷҜйғҪдјҡе……еҪ“дёҚеҗҢзҡ„и§’иүІпјҢеңЁе…¬еҸёдҪ е°ұжҳҜзү№е®ҡеІ—дҪҚзҡ„е‘ҳе·ҘпјҢеңЁе®¶еәӯдёӯдҪ е°ұжҳҜдёҖеҗҚе®¶еәӯжҲҗе‘ҳгҖӮйҡҸзқҖеңәжҷҜзҡ„еҲҮжҚўпјҢжҲ‘д»¬зҡ„и§’иүІд№ҹеңЁйҡҸд№ӢеҸҳеҢ–гҖӮ

жүҖд»ҘRBAC2еңЁRBAC0зҡ„еҹәзЎҖдёҠеј•е…ҘдәҶйқҷжҖҒиҒҢиҙЈеҲҶзҰ»пјҲStatic Separation of DutyпјҢз®Җз§°SSDпјүе’ҢеҠЁжҖҒиҒҢиҙЈеҲҶзҰ»пјҲDynamic Separation of DutyпјҢз®Җз§°DSDпјүдёӨдёӘзәҰжқҹжҰӮеҝөгҖӮд»–д»¬дёӨдёӘдҪңз”Ёзҡ„з”ҹе‘Ҫе‘ЁжңҹжҳҜдёҚеҗҢзҡ„;

  • SSDдҪңз”ЁдәҺзәҰжқҹз”ЁжҲ·е’Ңи§’иүІз»‘е®ҡж—¶гҖӮ 1.дә’ж–Ҙи§’иүІпјҡе°ұеғҸдёҠйқўзҡ„дҫӢеӯҗдҪ дёҚиғҪж—ўжҳҜAеҸҲжҳҜBпјҢдә’ж–Ҙзҡ„и§’иүІеҸӘиғҪдәҢйҖүдёҖ ; 2. ж•°йҮҸзәҰжқҹпјҡз”ЁжҲ·зҡ„и§’иүІж•°йҮҸжҳҜжңүйҷҗзҡ„дёҚиғҪеӨҡдәҺжҹҗдёӘеҹәж•°пјӣ 3. жқЎд»¶зәҰжқҹпјҡеҸӘиғҪиҫҫеҲ°жҹҗдёӘжқЎд»¶жүҚиғҪжӢҘжңүжҹҗдёӘи§’иүІгҖӮз»Ҹеёёз”ЁдәҺз”ЁжҲ·зӯүзә§дҪ“зі»пјҢеҸӘжңүдҪ е……й’ұжҲҗдёәVIPжүҚиғҪдёҖеҲҖ999гҖӮ
  • DSDдҪңз”ЁдәҺдјҡиҜқе’Ңи§’иүІдәӨдә’ж—¶гҖӮеҪ“з”ЁжҲ·жҢҒжңүеӨҡдёӘи§’иүІпјҢеңЁз”ЁжҲ·йҖҡиҝҮдјҡиҜқжҝҖжҙ»и§’иүІж—¶еҠ д»ҘжқЎд»¶зәҰжқҹпјҢж №жҚ®дёҚеҗҢзҡ„жқЎд»¶жү§иЎҢдёҚеҗҢзҡ„зӯ–з•ҘгҖӮ

3.4 RBAC3

жҲ‘е…ЁйғҪиҰҒ!

RBAC1е’ҢRBAC2еҗ„жңүзҘһйҖҡгҖӮеҪ“дҪ жӢҝзқҖиҝҷдёӨдёӘж–№жЎҲз»ҷдә§е“Ғз»ҸзҗҶзңӢж—¶пјҢд»–з»ҷдәҶдҪ дёҖдёӘеқҡе®ҡзҡ„зңјзҘһпјҡжҲ‘е…ЁйғҪиҰҒпјҒдәҺжҳҜRBAC3е°ұеҮәзҺ°дәҶгҖӮд№ҹе°ұжҳҜиҜҙRBAC3 = RBAC1 + RBAC2гҖӮ

4. RBAC дёӯдёҖдәӣжҰӮеҝөзҡ„зҗҶи§Ј

еӣӣдёӘжЁЎеһӢиҜҙе®ҢпјҢжҲ‘д»¬жқҘз®ҖеҚ•еҜ№е…¶дёӯзҡ„дёҖдәӣжҰӮеҝөиҝӣиЎҢиҝӣдёҖжӯҘзҡ„дәҶи§ЈгҖӮ

4.1 з”ЁжҲ·пјҲUserпјү

еҜ№з”ЁжҲ·зҡ„зҗҶи§ЈдёҚеә”иҜҘиў«еұҖйҷҗдәҺеҚ•дёӘз”ЁжҲ·пјҢд№ҹеҸҜд»ҘжҳҜз”ЁжҲ·з»„пјҲзұ»дјј linux зҡ„ User Groupпјү, жҲ–и®ёиҝҳжңүе…¶е®ғзҡ„еҗҚеӯ—жҜ”еҰӮйғЁй—ЁжҲ–иҖ…е…¬еҸёпјӣд№ҹеҸҜд»ҘжҳҜиҷҡжӢҹзҡ„иҙҰжҲ·пјҢе®ўжҲ·пјҢз”ҡиҮіиҜҙз¬¬дёүж–№еә”з”Ёд№ҹеҸҜд»Ҙз®—з”ЁжҲ· гҖӮжүҖд»ҘеҜ№з”ЁжҲ·зҡ„зҗҶи§ЈиҰҒе®ҪжіӣдёҖдәӣпјҢеҸӘиҰҒжҳҜжңүи®ҝй—®иө„жәҗйңҖжұӮзҡ„дё»дҪ“йғҪеҸҜд»Ҙзәіе…Ҙз”ЁжҲ·иҢғз•ҙгҖӮ

4.2 и§’иүІпјҲRoleпјү

и§’иүІжҳҜзү№е®ҡи®ёеҸҜзҡ„йӣҶеҗҲд»ҘеҸҠиҪҪдҪ“гҖӮдёҖдёӘи§’иүІеҸҜд»ҘеҢ…еҗ«еӨҡдёӘз”ЁжҲ·пјҢдёҖдёӘз”ЁжҲ·еҗҢж ·зҡ„д№ҹеҸҜд»ҘеұһдәҺеӨҡдёӘи§’иүІпјӣеҗҢж ·зҡ„дёҖдёӘи§’иүІеҸҜд»ҘеҢ…еҗ«еӨҡдёӘз”ЁжҲ·з»„пјҢдёҖдёӘз”ЁжҲ·з»„д№ҹеҸҜд»Ҙе…·жңүеӨҡдёӘи§’иүІпјҢжүҖд»Ҙи§’иүІе’Ңз”ЁжҲ·жҳҜеӨҡеҜ№еӨҡзҡ„е…ізі»гҖӮи§’иүІжҳҜеҸҜд»Ҙз»ҶеҲҶзҡ„пјҢд№ҹе°ұжҳҜеҸҜд»Ҙз»§жүҝгҖҒеҸҜд»ҘеҲҶз»„зҡ„гҖӮ

4.3 и®ёеҸҜпјҲPermissionпјү

и®ёеҸҜдёҖиҲ¬з§°е®ғдёәжқғйҷҗгҖӮйҖҡеёёжҲ‘д»¬е°Ҷи®ҝй—®зҡ„зӣ®ж Үз»ҹз§°дёәиө„жәҗпјҢдёҚз®ЎжҳҜж•°жҚ®иҝҳжҳҜйқҷжҖҒиө„жәҗйғҪжҳҜиө„жәҗгҖӮжҲ‘д»¬и®ҝй—®иө„жәҗеҹәжң¬дёҠеҸҲйҖҡиҝҮapiжҺҘеҸЈжқҘи®ҝй—®гҖӮжүҖд»ҘдёҖиҲ¬жқғйҷҗйғҪдҪ“зҺ°еңЁеҜ№жҺҘеҸЈзҡ„жҺ§еҲ¶дёҠгҖӮеҶҚз»ҶеҲҶзҡ„иҜқжҲ‘е°Ҷе…¶еҲ’еҲҶдёәиҸңеҚ•жҺ§еҲ¶пјҢе…·дҪ“ж•°жҚ®еўһеҲ ж”№жҹҘеҠҹиғҪжҺ§еҲ¶пјҲеүҚеҸ°дҪ“зҺ°дёәжҢүй’®пјүгҖӮеҸҰеӨ–и®ёеҸҜе…·жңүеҺҹеӯҗжҖ§пјҢдёҚеҸҜеҶҚеҲҶгҖӮжҲ‘д»¬е°Ҷи®ёеҸҜжҺҲдәҲи§’иүІж—¶е°ұжҳҜзІ’еәҰжңҖе°Ҹзҡ„еҚ•е…ғгҖӮ

5. жҖ»з»“

еҹәдәҺи§’иүІзҡ„и®ҝй—®жҺ§еҲ¶пјҲRBACпјүе·ІжҲҗдёәй«ҳзә§и®ҝй—®жҺ§еҲ¶зҡ„дё»иҰҒж–№жі•д№ӢдёҖгҖӮйҖҡиҝҮRBACпјҢжӮЁеҸҜд»ҘжҺ§еҲ¶жңҖз»Ҳз”ЁжҲ·еңЁе№ҝд№үе’ҢзІҫз»Ҷзә§еҲ«дёҠеҸҜд»ҘеҒҡд»Җд№ҲгҖӮжӮЁеҸҜд»ҘжҢҮе®ҡз”ЁжҲ·жҳҜз®ЎзҗҶе‘ҳпјҢдё“е®¶з”ЁжҲ·иҝҳжҳҜжңҖз»Ҳз”ЁжҲ·пјҢе№¶дҪҝи§’иүІе’Ңи®ҝй—®жқғйҷҗдёҺз»„з»Үдёӯе‘ҳе·Ҙзҡ„иҒҢдҪҚдҝқжҢҒдёҖиҮҙгҖӮд»…ж №жҚ®йңҖиҰҒдёәе‘ҳе·Ҙе®ҢжҲҗе·ҘдҪңзҡ„и¶іеӨҹи®ҝй—®жқғйҷҗжқҘеҲҶй…ҚжқғйҷҗгҖӮйҖҡиҝҮдёҠйқўзҡ„д»Ӣз»ҚзӣёдҝЎдёҖе®ҡдјҡи®©дҪ жңүжүҖж”¶иҺ·гҖӮе…¶е®һдёҚз®ЎдҪ дҪҝз”Ёд»Җд№Ҳе®үе…ЁжЎҶжһ¶пјҢ RBAC йғҪжҳҜеҝ…йЎ»жҺҢжҸЎзҡ„

2.1 HTTP Basic Authentication

HTTP Basic Authentication еҸҲеҸ«еҹәзЎҖи®ӨиҜҒпјҢе®ғз®ҖеҚ•ең°дҪҝз”Ё Base64 з®—жі•еҜ№з”ЁжҲ·еҗҚгҖҒеҜҶз ҒиҝӣиЎҢеҠ еҜҶпјҢе№¶е°ҶеҠ еҜҶеҗҺзҡ„дҝЎжҒҜж”ҫеңЁиҜ·жұӮеӨҙ Header дёӯпјҢжң¬иҙЁдёҠиҝҳжҳҜжҳҺж–Үдј иҫ“з”ЁжҲ·еҗҚгҖҒеҜҶз ҒпјҢе№¶дёҚе®үе…ЁпјҢжүҖд»ҘжңҖеҘҪеңЁ Https зҺҜеўғдёӢдҪҝз”ЁгҖӮе…¶и®ӨиҜҒжөҒзЁӢеҰӮдёӢпјҡ

е®ўжҲ·з«ҜеҸ‘иө· GET иҜ·жұӮ жңҚеҠЎз«Ҝе“Қеә”иҝ”еӣһ 401 UnauthorizedпјҢ www-Authenticate жҢҮе®ҡи®ӨиҜҒз®—жі•пјҢrealm жҢҮе®ҡе®үе…ЁеҹҹгҖӮз„¶еҗҺе®ўжҲ·з«ҜдёҖиҲ¬дјҡеј№зӘ—жҸҗзӨәиҫ“е…Ҙз”ЁжҲ·еҗҚз§°е’ҢеҜҶз ҒпјҢиҫ“е…Ҙз”ЁжҲ·еҗҚеҜҶз ҒеҗҺж”ҫе…Ҙ Header еҶҚж¬ЎиҜ·жұӮпјҢжңҚеҠЎз«Ҝи®ӨиҜҒжҲҗеҠҹеҗҺд»Ҙ 200 зҠ¶жҖҒз Ғе“Қеә”е®ўжҲ·з«ҜгҖӮ

2.2 HTTP Digest Authentication

дёәејҘиЎҘ BASIC и®ӨиҜҒеӯҳеңЁзҡ„ејұзӮ№е°ұжңүдәҶ HTTP Digest Authentication гҖӮе®ғеҸҲеҸ«ж‘ҳиҰҒи®ӨиҜҒгҖӮе®ғдҪҝз”ЁйҡҸжңәж•°еҠ дёҠ MD5 з®—жі•жқҘеҜ№з”ЁжҲ·еҗҚгҖҒеҜҶз ҒиҝӣиЎҢж‘ҳиҰҒзј–з ҒпјҢжөҒзЁӢзұ»дјј Http Basic Authentication пјҢдҪҶжҳҜжӣҙеҠ еӨҚжқӮдёҖдәӣ:

жӯҘйӘӨ1пјҡи·ҹеҹәзЎҖи®ӨиҜҒдёҖж ·пјҢеҸӘдёҚиҝҮиҝ”еӣһеёҰ WWW-Authenticate йҰ–йғЁеӯ—ж®өзҡ„е“Қеә”гҖӮиҜҘеӯ—ж®өеҶ…еҢ…еҗ«иҙЁй—®е“Қеә”ж–№ејҸи®ӨиҜҒжүҖйңҖиҰҒзҡ„дёҙж—¶е’ЁиҜўз ҒпјҲйҡҸжңәж•°пјҢnonceпјүгҖӮ йҰ–йғЁеӯ—ж®ө WWW-Authenticate еҶ…еҝ…йЎ»еҢ…еҗ« realm е’Ң nonce иҝҷдёӨдёӘеӯ—ж®өзҡ„дҝЎжҒҜгҖӮе®ўжҲ·з«Ҝе°ұжҳҜдҫқйқ еҗ‘жңҚеҠЎеҷЁеӣһйҖҒиҝҷдёӨдёӘеҖјиҝӣиЎҢи®ӨиҜҒзҡ„гҖӮnonce жҳҜдёҖз§ҚжҜҸж¬ЎйҡҸиҝ”еӣһзҡ„ 401 е“Қеә”з”ҹжҲҗзҡ„д»»ж„ҸйҡҸжңәеӯ—з¬ҰдёІгҖӮиҜҘеӯ—з¬ҰдёІйҖҡеёёжҺЁиҚҗз”ұ Base64 зј–з Ғзҡ„еҚҒе…ӯиҝӣеҲ¶ж•°зҡ„з»„жҲҗеҪўејҸпјҢдҪҶе®һйҷ…еҶ…е®№дҫқиө–жңҚеҠЎеҷЁзҡ„е…·дҪ“е®һзҺ°

жӯҘйӘӨ2пјҡжҺҘж”¶еҲ° 401 зҠ¶жҖҒз Ғзҡ„е®ўжҲ·з«ҜпјҢиҝ”еӣһзҡ„е“Қеә”дёӯеҢ…еҗ« DIGEST и®ӨиҜҒеҝ…йЎ»зҡ„йҰ–йғЁеӯ—ж®ө Authorization дҝЎжҒҜгҖӮйҰ–йғЁеӯ—ж®ө Authorization еҶ…еҝ…йЎ»еҢ…еҗ« usernameгҖҒrealmгҖҒnonceгҖҒuri е’Ң response зҡ„еӯ—ж®өдҝЎжҒҜпјҢе…¶дёӯпјҢrealm е’Ң nonce е°ұжҳҜд№ӢеүҚд»ҺжңҚеҠЎеҷЁжҺҘж”¶еҲ°зҡ„е“Қеә”дёӯзҡ„еӯ—ж®өгҖӮ

жӯҘйӘӨ3пјҡжҺҘж”¶еҲ°еҢ…еҗ«йҰ–йғЁеӯ—ж®ө Authorization иҜ·жұӮзҡ„жңҚеҠЎеҷЁпјҢдјҡзЎ®и®Өи®ӨиҜҒдҝЎжҒҜзҡ„жӯЈзЎ®жҖ§гҖӮи®ӨиҜҒйҖҡиҝҮеҗҺеҲҷдјҡиҝ”еӣһеҢ…еҗ« Request-URI иө„жәҗзҡ„е“Қеә”гҖӮ

2.3 SSL е®ўжҲ·з«Ҝи®ӨиҜҒ

SSL е®ўжҲ·з«Ҝи®ӨиҜҒе°ұжҳҜйҖҡеёёжҲ‘д»¬иҜҙзҡ„ HTTPS гҖӮе®үе…Ёзә§еҲ«иҫғй«ҳпјҢдҪҶйңҖиҰҒжүҝжӢ… CA иҜҒд№Ұиҙ№з”ЁгҖӮSSL и®ӨиҜҒиҝҮзЁӢдёӯж¶үеҸҠеҲ°дёҖдәӣйҮҚиҰҒзҡ„жҰӮеҝөпјҢж•°еӯ—иҜҒд№Ұжңәжһ„зҡ„е…¬й’ҘгҖҒиҜҒд№Ұзҡ„з§Ғй’Ҙе’Ңе…¬й’ҘгҖҒйқһеҜ№з§°з®—жі•пјҲй…ҚеҗҲиҜҒд№Ұзҡ„з§Ғй’Ҙе’Ңе…¬й’ҘдҪҝз”ЁпјүгҖҒеҜ№з§°еҜҶй’ҘгҖҒеҜ№з§°з®—жі•пјҲй…ҚеҗҲеҜ№з§°еҜҶй’ҘдҪҝз”ЁпјүгҖӮзӣёеҜ№еӨҚжқӮдёҖдәӣиҝҷйҮҢдёҚиҝҮеӨҡи®Іиҝ°гҖӮ

2.4 Form иЎЁеҚ•и®ӨиҜҒ

Form иЎЁеҚ•зҡ„и®ӨиҜҒж–№ејҸе№¶дёҚжҳҜHTTPи§„иҢғгҖӮжүҖд»Ҙе®һзҺ°ж–№ејҸд№ҹе‘ҲзҺ°еӨҡж ·еҢ–пјҢе…¶е®һжҲ‘д»¬е№іеёёзҡ„жү«з Ғзҷ»еҪ•пјҢжүӢжңәйӘҢиҜҒз Ғзҷ»еҪ•йғҪеұһдәҺиЎЁеҚ•зҷ»еҪ•зҡ„иҢғз•ҙгҖӮиЎЁеҚ•и®ӨиҜҒдёҖиҲ¬йғҪдјҡй…ҚеҗҲ CookieпјҢSession зҡ„дҪҝз”ЁпјҢзҺ°еңЁеҫҲеӨҡ Web з«ҷзӮ№йғҪдҪҝз”ЁжӯӨи®ӨиҜҒж–№ејҸгҖӮз”ЁжҲ·еңЁзҷ»еҪ•йЎөдёӯеЎ«еҶҷз”ЁжҲ·еҗҚе’ҢеҜҶз ҒпјҢжңҚеҠЎз«Ҝи®ӨиҜҒйҖҡиҝҮеҗҺдјҡе°Ҷ sessionId иҝ”еӣһз»ҷжөҸи§ҲеҷЁз«ҜпјҢжөҸи§ҲеҷЁдјҡдҝқеӯҳ sessionId еҲ°жөҸи§ҲеҷЁзҡ„ Cookie дёӯгҖӮеӣ дёә HTTP жҳҜж— зҠ¶жҖҒзҡ„пјҢжүҖд»ҘжөҸи§ҲеҷЁдҪҝз”Ё Cookie жқҘдҝқеӯҳ sessionIdгҖӮдёӢж¬Ўе®ўжҲ·з«ҜдјҡеңЁеҸ‘йҖҒзҡ„иҜ·жұӮдёӯдјҡжҗәеёҰ sessionId еҖјпјҢжңҚеҠЎз«ҜеҸ‘зҺ° sessionId еӯҳеңЁе№¶д»ҘжӯӨдёәзҙўеј•иҺ·еҸ–з”ЁжҲ·еӯҳеңЁжңҚеҠЎз«Ҝзҡ„и®ӨиҜҒдҝЎжҒҜиҝӣиЎҢи®ӨиҜҒж“ҚдҪңгҖӮи®ӨиҜҒиҝҮеҲҷдјҡжҸҗдҫӣиө„жәҗи®ҝй—®гҖӮ

жҲ‘д»¬еңЁSpring Security10 - зҷ»еҪ•еҗҺиҝ”еӣһ JWT Token дёҖж–Үе…¶е®һд№ҹжҳҜйҖҡиҝҮ Form жҸҗдәӨжқҘиҺ·еҸ– Jwt е…¶е®һ Jwt и·ҹ sessionId еҗҢж ·зҡ„дҪңз”ЁпјҢеҸӘдёҚиҝҮ Jwt еӨ©з„¶жҗәеёҰдәҶз”ЁжҲ·зҡ„дёҖдәӣдҝЎжҒҜпјҢиҖҢ sessionId йңҖиҰҒеҺ»иҝӣдёҖжӯҘиҺ·еҸ–з”ЁжҲ·дҝЎжҒҜгҖӮ

2.5 Json Web Token зҡ„и®ӨиҜҒж–№ејҸ Bearer Authentication

жҲ‘д»¬йҖҡиҝҮиЎЁеҚ•и®ӨиҜҒиҺ·еҸ– Json Web Token пјҢйӮЈд№ҲеҰӮдҪ•дҪҝз”Ёе®ғе‘ў? йҖҡеёёжҲ‘д»¬дјҡжҠҠ Jwt дҪңдёәд»ӨзүҢдҪҝз”Ё Bearer Authentication ж–№ејҸдҪҝз”ЁгҖӮBearer Authentication жҳҜдёҖз§ҚеҹәдәҺд»ӨзүҢзҡ„ HTTP иә«д»ҪйӘҢиҜҒж–№жЎҲпјҢз”ЁжҲ·еҗ‘жңҚеҠЎеҷЁиҜ·жұӮи®ҝй—®еҸ—йҷҗиө„жәҗж—¶пјҢдјҡжҗәеёҰдёҖдёӘ Token дҪңдёәеҮӯиҜҒпјҢжЈҖйӘҢйҖҡиҝҮеҲҷеҸҜд»Ҙи®ҝй—®зү№е®ҡзҡ„иө„жәҗгҖӮжңҖеҲқжҳҜеңЁ RFC 6750 дёӯдҪңдёә OAuth 2.0 зҡ„дёҖйғЁеҲҶпјҢдҪҶжңүж—¶д№ҹеҸҜд»ҘеҚ•зӢ¬дҪҝз”ЁгҖӮ
жҲ‘д»¬еңЁдҪҝз”Ё Bear Token зҡ„ж–№жі•жҳҜеңЁиҜ·жұӮеӨҙзҡ„ Authorization еӯ—ж®өдёӯж”ҫе…Ҙ Bearer <token> зҡ„ж јејҸзҡ„еҠ еҜҶдёІ(Json Web Token)гҖӮиҜ·жіЁж„Ҹ Bearer еүҚзјҖдёҺ Token д№Ӣй—ҙжңүдёҖдёӘз©әеӯ—з¬ҰдҪҚпјҢдёҺеҹәжң¬иә«д»ҪйӘҢиҜҒзұ»дјјпјҢBearer Authentication еҸӘиғҪеңЁHTTPSпјҲSSLпјүдёҠдҪҝз”ЁгҖӮ

AuthenticationException жҳҜеңЁз”ЁжҲ·и®ӨиҜҒзҡ„ж—¶еҖҷеҮәзҺ°й”ҷиҜҜж—¶жҠӣеҮәзҡ„ејӮеёёгҖӮдё»иҰҒзҡ„еӯҗзұ»еҰӮеӣҫпјҡ

AccessDeniedException дё»иҰҒжҳҜеңЁз”ЁжҲ·еңЁи®ҝй—®еҸ—дҝқжҠӨиө„жәҗж—¶иў«жӢ’з»қиҖҢжҠӣеҮәзҡ„ејӮеёёгҖӮеҗҢAuthenticationException дёҖж ·е®ғд№ҹжҸҗдҫӣдәҶдёҖдәӣе…·дҪ“зҡ„еӯҗзұ»гҖӮеҰӮдёӢеӣҫпјҡ

Posted on 2021-07-06 In Spring Security

1. еүҚиЁҖ

дёҠдёҖзҜҮеҜ№ Spring Security жүҖжңүеҶ…зҪ®зҡ„ Filter иҝӣиЎҢдәҶд»Ӣз»ҚгҖӮд»ҠеӨ©жҲ‘д»¬жқҘе®һжҲҳеҰӮдҪ•е®үе…ЁйҖҖеҮәеә”з”ЁзЁӢеәҸгҖӮ

2. жҲ‘д»¬дҪҝз”Ё Spring Security зҷ»еҪ•еҗҺйғҪеҒҡдәҶд»Җд№Ҳ

иҝҷдёӘй—®йўҳжҲ‘д»¬еҝ…йЎ»жҗһжё…жҘҡпјҒдёҖиҲ¬зҷ»еҪ•еҗҺпјҢжңҚеҠЎз«Ҝдјҡз»ҷз”ЁжҲ·еҸ‘дёҖдёӘеҮӯиҜҒгҖӮеёёи§Ғжңүд»ҘдёӢзҡ„дёӨз§Қпјҡ

  • еҹәдәҺ Session е®ўжҲ·з«Ҝдјҡеӯҳ cookie жқҘдҝқеӯҳдёҖдёӘ sessionId пјҢжңҚеҠЎз«ҜеӯҳдёҖдёӘ Session гҖӮ
  • еҹәдәҺ token е®ўжҲ·з«ҜеӯҳдёҖдёӘ token дёІпјҢжңҚеҠЎз«ҜдјҡеңЁзј“еӯҳдёӯеӯҳдёҖдёӘз”ЁжқҘж ЎйӘҢжӯӨ token зҡ„дҝЎжҒҜгҖӮ

3. йҖҖеҮәзҷ»еҪ•йңҖиҰҒжҲ‘д»¬еҒҡд»Җд№Ҳ

  1. еҪ“еүҚзҡ„з”ЁжҲ·зҷ»еҪ•зҠ¶жҖҒеӨұж•ҲгҖӮиҝҷе°ұйңҖиҰҒжҲ‘д»¬жё…йҷӨжңҚеҠЎз«Ҝзҡ„з”ЁжҲ·зҠ¶жҖҒгҖӮ
  2. йҖҖеҮәзҷ»еҪ•жҺҘеҸЈе№¶дёҚжҳҜ permitAllпјҢ еҸӘжңүжҗәеёҰеҜ№еә”з”ЁжҲ·зҡ„еҮӯиҜҒжүҚйҖҖеҮәгҖӮ
  3. е°ҶйҖҖеҮәз»“жһңиҝ”еӣһз»ҷиҜ·жұӮж–№гҖӮ
  4. йҖҖеҮәзҷ»еҪ•еҗҺз”ЁжҲ·еҸҜд»ҘйҖҡиҝҮйҮҚж–°зҷ»еҪ•жқҘи®ӨиҜҒиҜҘз”ЁжҲ·гҖӮ

4. Spring Security дёӯзҡ„йҖҖеҮәзҷ»еҪ•

жҺҘдёӢжқҘжҲ‘д»¬жқҘеҲҶжһҗе№¶е®һжҲҳ еҰӮдҪ•е®ҡеҲ¶йҖҖеҮәзҷ»еҪ•йҖ»иҫ‘гҖӮйҰ–е…ҲжҲ‘д»¬иҰҒдәҶи§Ј LogoutFilter гҖӮ

4.1 LogoutFilter

йҖҡиҝҮеүҚж–ҮжҲ‘д»¬зҹҘйҒ“йҖҖеҮәзҷ»еҪ•йҖ»иҫ‘жҳҜз”ұиҝҮж»ӨеҷЁ LogoutFilter жқҘжү§иЎҢзҡ„гҖӮ е®ғжҢҒжңүдёүдёӘжҺҘеҸЈзұ»еһӢзҡ„еұһжҖ§пјҡ

  1. RequestMatcher logoutRequestMatcher иҝҷдёӘз”ЁжқҘжӢҰжҲӘйҖҖеҮәиҜ·жұӮзҡ„ URL
  2. LogoutHandler handler з”ЁжқҘеӨ„зҗҶйҖҖеҮәзҡ„е…·дҪ“йҖ»иҫ‘
  3. LogoutSuccessHandler logoutSuccessHandler йҖҖеҮәжҲҗеҠҹеҗҺжү§иЎҢзҡ„йҖ»иҫ‘

жҲ‘д»¬йҖҡиҝҮеҜ№д»ҘдёҠдёүдёӘжҺҘеҸЈзҡ„е®һзҺ°е°ұиғҪе®һзҺ°жҲ‘д»¬иҮӘе®ҡд№үзҡ„йҖҖеҮәйҖ»иҫ‘гҖӮ

4.2 LogoutConfigurer

жҲ‘д»¬дёҖиҲ¬дёҚдјҡзӣҙжҺҘж“ҚдҪң LogoutFilter пјҢиҖҢжҳҜйҖҡиҝҮ LogoutConfigurer жқҘй…ҚзҪ® LogoutFilterгҖӮ дҪ еҸҜд»ҘйҖҡиҝҮ HttpSecurity.logout() ж–№жі•жқҘеҲқе§ӢеҢ–дёҖдёӘ LogoutConfigurer гҖӮ жҺҘдёӢжқҘжҲ‘д»¬жқҘе®һжҲҳж“ҚдҪңдёҖдёӢгҖӮ

4.2.1 е®һзҺ°иҮӘе®ҡд№үйҖҖеҮәзҷ»еҪ•иҜ·жұӮURL

LogoutConfigurer жҸҗдҫӣдәҶ logoutRequestMatcher(RequestMatcher logoutRequestMatcher)гҖҒlogoutUrl(Sring logoutUrl) дёӨз§Қж–№ејҸжқҘе®ҡд№үйҖҖеҮәзҷ»еҪ•иҜ·жұӮзҡ„ URL гҖӮе®ғд»¬дҪңз”ЁжҳҜзӣёеҗҢзҡ„пјҢдҪ йҖүжӢ©е…¶дёӯдёҖз§Қж–№ејҸеҚіеҸҜгҖӮ

4.2.2 еӨ„зҗҶе…·дҪ“зҡ„йҖ»иҫ‘

й»ҳи®Өжғ…еҶөдёӢ Spring Security жҳҜеҹәдәҺ Session зҡ„гҖӮLogoutConfigurer жҸҗдҫӣдәҶдёҖдәӣзӣҙжҺҘй…ҚзҪ®жқҘж»Ўи¶ідҪ зҡ„йңҖиҰҒгҖӮеҰӮдёӢпјҡ

  • clearAuthentication(boolean clearAuthentication) жҳҜеҗҰеңЁйҖҖеҮәж—¶жё…йҷӨеҪ“еүҚз”ЁжҲ·зҡ„и®ӨиҜҒдҝЎжҒҜ
  • deleteCookies(String... cookieNamesToClear) еҲ йҷӨжҢҮе®ҡзҡ„ cookies
  • invalidateHttpSession(boolean invalidateHttpSession) жҳҜеҗҰз§»йҷӨ HttpSession

еҰӮжһңдёҠйқўж»Ўи¶ідёҚдәҶдҪ зҡ„йңҖиҰҒе°ұйңҖиҰҒдҪ жқҘе®ҡеҲ¶ LogoutHandler дәҶгҖӮ

4.2.3 йҖҖеҮәжҲҗеҠҹйҖ»иҫ‘

  • logoutSuccessUrl(String logoutSuccessUrl) йҖҖеҮәжҲҗеҠҹеҗҺдјҡиў«йҮҚе®ҡеҗ‘еҲ°жӯӨ URL пјҢдҪ еҸҜд»ҘеҶҷдёҖдёӘController жқҘе®ҢжҲҗжңҖз»Ҳиҝ”еӣһпјҢдҪҶжҳҜйңҖиҰҒж”ҜжҢҒ GET иҜ·жұӮе’Ң еҢҝеҗҚи®ҝй—® гҖӮ йҖҡиҝҮ setDefaultTargetUrl ж–№жі•жіЁе…ҘеҲ° LogoutSuccessHandler
  • defaultLogoutSuccessHandlerFor(LogoutSuccessHandler handler, RequestMatcher preferredMatcher) з”ЁжқҘжһ„йҖ й»ҳи®Өзҡ„ LogoutSuccessHandler жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮж·»еҠ еӨҡдёӘжқҘе®һзҺ°д»ҺдёҚеҗҢ URL йҖҖеҮәжү§иЎҢдёҚеҗҢзҡ„йҖ»иҫ‘гҖӮ
  • LogoutSuccessHandler logoutSuccessHandler йҖҖеҮәжҲҗеҠҹеҗҺжү§иЎҢзҡ„йҖ»иҫ‘зҡ„жҠҪиұЎж №жң¬жҺҘеҸЈгҖӮ

4.3 Spring Security йҖҖеҮәзҷ»еҪ•е®һжҲҳ

зҺ°еңЁеүҚеҗҺз«ҜеҲҶзҰ»жҜ”иҫғеӨҡпјҢйҖҖеҮәеҗҺиҝ”еӣһjsonгҖӮ иҖҢдё”еҸӘжңүз”ЁжҲ·еңЁзәҝжүҚиғҪйҖҖеҮәзҷ»еҪ•гҖӮеҗҰеҲҷдёҚиғҪиҝӣиЎҢйҖҖеҮәж“ҚдҪңгҖӮжҲ‘д»¬йҮҮз”Ёе®һзҺ° LogoutHandler е’Ң LogoutSuccessHandler жҺҘеҸЈиҝҷз§Қзј–зЁӢзҡ„ж–№ејҸжқҘй…ҚзҪ® гҖӮйҖҖеҮәиҜ·жұӮзҡ„ url дҫқз„¶йҖҡиҝҮ LogoutConfigurer.logoutUrl(String logoutUrl)жқҘе®ҡд№үгҖӮ

й»ҳи®Өжғ…еҶөдёӢжё…йҷӨи®ӨиҜҒдҝЎжҒҜ пјҲinvalidateHttpSessionпјүпјҢе’ҢSession еӨұж•ҲпјҲinvalidateHttpSessionпјү е·Із»Ҹз”ұеҶ…зҪ®зҡ„SecurityContextLogoutHandlerжқҘе®ҢжҲҗгҖӮжҲ‘д»¬иҮӘе®ҡд№үзҡ„LogoutHandler дјҡеңЁSecurityContextLogoutHandler` жқҘжү§иЎҢгҖӮ

4.3.2 иҮӘе®ҡд№ү LogoutSuccessHandler

еҰӮжһңжҲ‘д»¬е®һзҺ°дәҶиҮӘе®ҡд№үзҡ„ LogoutSuccessHandler е°ұдёҚеҝ…иҰҒи®ҫзҪ® LogoutConfigurer#logoutSuccessUrl(String logoutSuccessUrl) дәҶгҖӮиҜҘеӨ„зҗҶеҷЁеӨ„зҗҶеҗҺдјҡе“Қеә”з»ҷеүҚз«ҜгҖӮдҪ еҸҜд»ҘиҪ¬еҸ‘еҲ°е…¶е®ғжҺ§еҲ¶еҷЁгҖӮйҮҚе®ҡеҗ‘еҲ°зҷ»еҪ•йЎөйқўпјҢд№ҹеҸҜд»ҘиҮӘиЎҢе®һзҺ°е…¶е®ғ MediaType ,еҸҜд»ҘжҳҜ json жҲ–иҖ…йЎөйқў

Spring Security еҶ…зҪ®дәҶдёҖдәӣиҝҮж»ӨеҷЁпјҢд»–д»¬еҗ„жңүеҗ„зҡ„жң¬дәӢгҖӮеҰӮжһңдҪ жҺҢжҸЎдәҶиҝҷдәӣиҝҮж»ӨеҷЁпјҢеҫҲеӨҡе®һйҷ…ејҖеҸ‘дёӯзҡ„йңҖжұӮе’Ңй—®йўҳйғҪеҫҲе®№жҳ“и§ЈеҶігҖӮд»ҠеӨ©жҲ‘д»¬жқҘи§ҒиҜҶдёҖдёӢиҝҷдәӣеҶ…зҪ®зҡ„иҝҮж»ӨеҷЁгҖӮ

2. еҶ…зҪ®иҝҮж»ӨеҷЁеҲқе§ӢеҢ–

еңЁ Spring Security еҲқе§ӢеҢ–ж ёеҝғиҝҮж»ӨеҷЁж—¶ HttpSecurity дјҡйҖҡиҝҮе°Ҷ Spring Security еҶ…зҪ®зҡ„дёҖдәӣиҝҮж»ӨеҷЁд»Ҙ FilterComparator жҸҗдҫӣзҡ„и§„еҲҷиҝӣиЎҢжҜ”иҫғжҢүз…§жҜ”иҫғз»“жһңиҝӣиЎҢжҺ’еәҸжіЁеҶҢгҖӮ

2.1 жҺ’еәҸи§„еҲҷ

FilterComparator з»ҙжҠӨдәҶдёҖдёӘйЎәеәҸзҡ„жіЁеҶҢиЎЁ filterToOrder гҖӮ

иҝҷдәӣе°ұжҳҜжүҖжңүеҶ…зҪ®зҡ„иҝҮж»ӨеҷЁгҖӮд»–д»¬жҳҜйҖҡиҝҮдёӢйқўзҡ„ж–№жі•иҺ·еҸ–иҮӘе·ұзҡ„еәҸеҸ·пјҡ

йҖҡиҝҮиҝҮж»ӨеҷЁзҡ„зұ»е…Ёйҷҗе®ҡеҗҚд»ҺжіЁеҶҢиЎЁ filterToOrder дёӯиҺ·еҸ–иҮӘе·ұзҡ„еәҸеҸ·пјҢеҰӮжһңжІЎжңүзӣҙжҺҘиҺ·еҸ–еҲ°еәҸеҸ·йҖҡиҝҮйҖ’еҪ’иҺ·еҸ–зҲ¶зұ»еңЁжіЁеҶҢиЎЁдёӯзҡ„еәҸеҸ·дҪңдёәиҮӘе·ұзҡ„еәҸеҸ·пјҢеәҸеҸ·и¶Ҡе°Ҹдјҳе…Ҳзә§и¶Ҡй«ҳгҖӮдёҠйқўзҡ„иҝҮж»ӨеҷЁе№¶йқһе…ЁйғЁдјҡиў«еҲқе§ӢеҢ–гҖӮжңүзҡ„йңҖиҰҒйўқеӨ–еј•е…ҘдёҖдәӣеҠҹиғҪеҢ…пјҢжңүзҡ„зңӢ HttpSecurity зҡ„й…ҚзҪ®жғ…еҶөгҖӮ еңЁдёҠдёҖзҜҮж–Үз« дёӯгҖӮжҲ‘д»¬зҰҒз”ЁдәҶ CSRF еҠҹиғҪпјҢе°ұж„Ҹе‘ізқҖ CsrfFilter дёҚдјҡиў«жіЁеҶҢгҖӮ

3. еҶ…зҪ®иҝҮж»ӨеҷЁи®Іи§Ј

жҺҘдёӢжқҘжҲ‘д»¬е°ұеҜ№иҝҷдәӣеҶ…зҪ®иҝҮж»ӨеҷЁиҝӣиЎҢдёҖдёӘзі»з»ҹзҡ„и®ӨиҜҶгҖӮжҲ‘д»¬е°ҶжҢүз…§й»ҳи®ӨйЎәеәҸиҝӣиЎҢи®Іи§ЈгҖӮ

3.1 ChannelProcessingFilter

ChannelProcessingFilter йҖҡеёёжҳҜз”ЁжқҘиҝҮж»Өе“ӘдәӣиҜ·жұӮеҝ…йЎ»з”Ё https еҚҸи®®пјҢ е“ӘдәӣиҜ·жұӮеҝ…йЎ»з”Ё http еҚҸи®®пјҢ е“ӘдәӣиҜ·жұӮйҡҸдҫҝз”Ёе“ӘдёӘеҚҸи®®йғҪиЎҢгҖӮе®ғдё»иҰҒжңүдёӨдёӘеұһжҖ§пјҡ

  • ChannelDecisionManager з”ЁжқҘеҲӨж–ӯиҜ·жұӮжҳҜеҗҰз¬ҰеҗҲж—ўе®ҡзҡ„еҚҸи®®и§„еҲҷгҖӮе®ғз»ҙжҠӨдәҶдёҖдёӘ ChannelProcessor еҲ—иЎЁ иҝҷдәӣChannelProcessor жҳҜе…·дҪ“з”ЁжқҘжү§иЎҢ ANY_CHANNEL зӯ–з•Ҙ пјҲд»»дҪ•йҖҡйҒ“йғҪеҸҜд»Ҙпјү, REQUIRES_SECURE_CHANNEL зӯ–з•Ҙ пјҲеҸӘиғҪйҖҡиҝҮhttpsйҖҡйҒ“пјү, REQUIRES_INSECURE_CHANNEL зӯ–з•Ҙ пјҲеҸӘиғҪйҖҡиҝҮ http йҖҡйҒ“пјүгҖӮ

  • FilterInvocationSecurityMetadataSource з”ЁжқҘеӯҳеӮЁ url дёҺ еҜ№еә”зҡ„ANY_CHANNELгҖҒREQUIRES_SECURE_CHANNELгҖҒREQUIRES_INSECURE_CHANNEL зҡ„жҳ е°„е…ізі»гҖӮ

ChannelProcessingFilterйҖҡиҝҮ HttpScurity#requiresChannel() зӯүзӣёе…іж–№жі•еј•е…Ҙе…¶й…ҚзҪ®еҜ№иұЎ ChannelSecurityConfigurer жқҘиҝӣиЎҢй…ҚзҪ®гҖӮ

3.2 ConcurrentSessionFilter

ConcurrentSessionFilter дё»иҰҒз”ЁжқҘеҲӨж–ӯsessionжҳҜеҗҰиҝҮжңҹд»ҘеҸҠжӣҙж–°жңҖж–°зҡ„и®ҝй—®ж—¶й—ҙгҖӮе…¶жөҒзЁӢдёәпјҡ

  1. session жЈҖжөӢпјҢеҰӮжһңдёҚеӯҳеңЁзӣҙжҺҘж”ҫиЎҢеҺ»жү§иЎҢдёӢдёҖдёӘиҝҮж»ӨеҷЁгҖӮеӯҳеңЁеҲҷиҝӣиЎҢдёӢдёҖжӯҘгҖӮ
  2. ж №жҚ®sessionidд»ҺSessionRegistryдёӯиҺ·еҸ–SessionInformationпјҢд»ҺSessionInformationдёӯиҺ·еҸ–sessionжҳҜеҗҰиҝҮжңҹпјӣжІЎжңүиҝҮжңҹеҲҷжӣҙж–°SessionInformationдёӯзҡ„и®ҝй—®ж—Ҙжңҹпјӣ
    еҰӮжһңиҝҮжңҹпјҢеҲҷжү§иЎҢdoLogout()ж–№жі•пјҢиҝҷдёӘж–№жі•дјҡе°Ҷsessionж— ж•ҲпјҢе№¶е°Ҷ SecurityContext дёӯзҡ„Authenticationдёӯзҡ„жқғйҷҗзҪ®з©әпјҢеҗҢж—¶еңЁSecurityContenxtHoloderдёӯжё…йҷӨSecurityContextз„¶еҗҺжҹҘзңӢжҳҜеҗҰжңүи·іиҪ¬зҡ„ expiredUrlпјҢеҰӮжһңжңүе°ұи·іиҪ¬пјҢжІЎжңүе°ұиҫ“еҮәжҸҗзӨәдҝЎжҒҜгҖӮ

ConcurrentSessionFilter йҖҡиҝҮSessionManagementConfigurer жқҘиҝӣиЎҢй…ҚзҪ®гҖӮ

3.3 WebAsyncManagerIntegrationFilter

WebAsyncManagerIntegrationFilterз”ЁдәҺйӣҶжҲҗSecurityContextеҲ°SpringејӮжӯҘжү§иЎҢжңәеҲ¶дёӯзҡ„WebAsyncManagerгҖӮз”ЁжқҘеӨ„зҗҶејӮжӯҘиҜ·жұӮзҡ„е®үе…ЁдёҠдёӢж–ҮгҖӮе…·дҪ“йҖ»иҫ‘дёәпјҡ

  1. д»ҺиҜ·жұӮеұһжҖ§дёҠиҺ·еҸ–жүҖз»‘е®ҡзҡ„WebAsyncManagerпјҢеҰӮжһңе°ҡжңӘз»‘е®ҡпјҢе…ҲеҒҡз»‘е®ҡгҖӮ
  2. д»ҺasyncManagerдёӯиҺ·еҸ– key дёә CALLABLE_INTERCEPTOR_KEY зҡ„е®үе…ЁдёҠдёӢж–ҮеӨҡзәҝзЁӢеӨ„зҗҶеҷЁ SecurityContextCallableProcessingInterceptor, еҰӮжһңиҺ·еҸ–еҲ°зҡ„дёә nullпјҢ
    ж–°е»әдёҖдёӘ SecurityContextCallableProcessingInterceptor е№¶з»‘е®ҡ CALLABLE_INTERCEPTOR_KEY жіЁеҶҢеҲ° asyncManager дёӯгҖӮ

иҝҷйҮҢз®ҖеҚ•иҜҙдёҖдёӢ SecurityContextCallableProcessingInterceptor гҖӮе®ғе®һзҺ°дәҶжҺҘеҸЈ CallableProcessingInterceptorпјҢ
еҪ“е®ғиў«еә”з”ЁдәҺдёҖж¬ЎејӮжӯҘжү§иЎҢж—¶пјҢbeforeConcurrentHandling() ж–№жі•дјҡеңЁи°ғз”ЁиҖ…зәҝзЁӢжү§иЎҢпјҢиҜҘж–№жі•дјҡзӣёеә”ең°д»ҺеҪ“еүҚзәҝзЁӢиҺ·еҸ–SecurityContext,з„¶еҗҺиў«и°ғз”ЁиҖ…зәҝзЁӢдёӯжү§иЎҢйҖ»иҫ‘ж—¶пјҢдјҡдҪҝз”ЁиҝҷдёӘ SecurityContextпјҢд»ҺиҖҢе®һзҺ°е®үе…ЁдёҠдёӢж–Үд»Һи°ғз”ЁиҖ…зәҝзЁӢеҲ°иў«и°ғз”ЁиҖ…зәҝзЁӢзҡ„дј иҫ“гҖӮ

WebAsyncManagerIntegrationFilter йҖҡиҝҮ WebSecurityConfigurerAdapter#getHttp()ж–№жі•ж·»еҠ еҲ° HttpSecurity дёӯжҲҗдёә DefaultSecurityFilterChain зҡ„дёҖдёӘй“ҫиҠӮгҖӮ

3.4 SecurityContextPersistenceFilter

SecurityContextPersistenceFilter дё»иҰҒжҺ§еҲ¶ SecurityContext зҡ„еңЁдёҖж¬ЎиҜ·жұӮдёӯзҡ„з”ҹе‘Ҫе‘Ёжңҹ гҖӮ иҜ·жұӮжқҘдёҙж—¶пјҢеҲӣе»әSecurityContext е®үе…ЁдёҠдёӢж–ҮдҝЎжҒҜпјҢиҜ·жұӮз»“жқҹж—¶жё…з©ә SecurityContextHolderгҖӮ

SecurityContextPersistenceFilter йҖҡиҝҮ HttpScurity#securityContext() еҸҠзӣёе…іж–№жі•еј•е…Ҙе…¶й…ҚзҪ®еҜ№иұЎ SecurityContextConfigurer жқҘиҝӣиЎҢй…ҚзҪ®гҖӮ

3.5 HeaderWriterFilter

HeaderWriterFilter з”ЁжқҘз»ҷ http е“Қеә”ж·»еҠ дёҖдәӣ Header,жҜ”еҰӮ X-Frame-Options, X-XSS-Protection пјҢX-Content-Type-OptionsгҖӮ

дҪ еҸҜд»ҘйҖҡиҝҮ HttpScurity#headers() жқҘе®ҡеҲ¶иҜ·жұӮHeader гҖӮ

3.6 CorsFilter

и·Ёеҹҹзӣёе…ізҡ„иҝҮж»ӨеҷЁгҖӮиҝҷжҳҜSpring MVC Javaй…ҚзҪ®е’ҢXML е‘ҪеҗҚз©әй—ҙ CORS й…ҚзҪ®зҡ„жӣҝд»Јж–№жі•пјҢ д»…еҜ№дҫқиө–дәҺspring-webзҡ„еә”з”ЁзЁӢеәҸжңүз”ЁпјҲдёҚйҖӮз”ЁдәҺspring-webmvcпјүжҲ– иҰҒжұӮеңЁjavax.servlet.Filter зә§еҲ«иҝӣиЎҢCORSжЈҖжҹҘзҡ„е®үе…ЁзәҰжқҹй“ҫжҺҘгҖӮиҝҷдёӘжҳҜзӣ®еүҚе®ҳж–№зҡ„дёҖдәӣи§ЈиҜ»пјҢдҪҶжҳҜжҲ‘иҝҳжҳҜдёҚеӨӘжё…жҘҡе®һйҷ…жңәеҲ¶гҖӮ

дҪ еҸҜд»ҘйҖҡиҝҮ HttpSecurity#cors() жқҘе®ҡеҲ¶гҖӮ

3.7 CsrfFilter

CsrfFilter з”ЁдәҺйҳІжӯўcsrfж”»еҮ»пјҢеүҚеҗҺз«ҜдҪҝз”ЁjsonдәӨдә’йңҖиҰҒжіЁж„Ҹзҡ„дёҖдёӘй—®йўҳгҖӮ

дҪ еҸҜд»ҘйҖҡиҝҮ HttpSecurity.csrf() жқҘејҖеҗҜжҲ–иҖ…е…ій—ӯе®ғгҖӮеңЁдҪ дҪҝз”Ё jwt зӯү token жҠҖжңҜж—¶пјҢжҳҜдёҚйңҖиҰҒиҝҷдёӘзҡ„гҖӮ

3.8 LogoutFilter

LogoutFilter еҫҲжҳҺжҳҫиҝҷжҳҜеӨ„зҗҶжіЁй”Җзҡ„иҝҮж»ӨеҷЁгҖӮ

дҪ еҸҜд»ҘйҖҡиҝҮ HttpSecurity.logout() жқҘе®ҡеҲ¶жіЁй”ҖйҖ»иҫ‘пјҢйқһеёёжңүз”ЁгҖӮ

3.9 OAuth2AuthorizationRequestRedirectFilter

е’ҢдёҠйқўзҡ„жңүжүҖдёҚеҗҢпјҢиҝҷдёӘйңҖиҰҒдҫқиө– spring-scurity-oauth2 зӣёе…ізҡ„жЁЎеқ—гҖӮиҜҘиҝҮж»ӨеҷЁжҳҜеӨ„зҗҶ OAuth2 иҜ·жұӮйҰ–йҖүйҮҚе®ҡеҗ‘зӣёе…ійҖ»иҫ‘зҡ„гҖӮд»ҘеҗҺдјҡжҲ‘дјҡеёҰдҪ д»¬и®ӨиҜҶе®ғпјҢиҜ·еӨҡеӨҡе…іжіЁе…¬дј—еҸ·пјҡFelordcn гҖӮ

3.10 Saml2WebSsoAuthenticationRequestFilter

иҝҷдёӘйңҖиҰҒз”ЁеҲ° Spring Security SAML жЁЎеқ—пјҢиҝҷжҳҜдёҖдёӘеҹәдәҺ SMAL зҡ„ SSO еҚ•зӮ№зҷ»еҪ•иҜ·жұӮи®ӨиҜҒиҝҮж»ӨеҷЁгҖӮ

е…ідәҺSAM

SAML еҚіе®үе…Ёж–ӯиЁҖж Үи®°иҜӯиЁҖпјҢиӢұж–Үе…Ёз§°жҳҜ Security Assertion Markup LanguageгҖӮе®ғжҳҜдёҖдёӘеҹәдәҺ XML зҡ„ж ҮеҮҶпјҢз”ЁдәҺеңЁдёҚеҗҢзҡ„е®үе…ЁеҹҹпјҲsecurity domainпјүд№Ӣй—ҙдәӨжҚўи®ӨиҜҒе’ҢжҺҲжқғж•°жҚ®гҖӮеңЁ SAML ж ҮеҮҶе®ҡд№үдәҶиә«д»ҪжҸҗдҫӣиҖ… (identity provider) е’ҢжңҚеҠЎжҸҗдҫӣиҖ… (service provider)пјҢиҝҷдёӨиҖ…жһ„жҲҗдәҶеүҚйқўжүҖиҜҙзҡ„дёҚеҗҢзҡ„е®үе…ЁеҹҹгҖӮ SAML жҳҜ OASIS з»„з»Үе®үе…ЁжңҚеҠЎжҠҖжңҜе§”е‘ҳдјҡ(Security Services Technical Committee) зҡ„дә§е“ҒгҖӮ

SAMLпјҲSecurity Assertion Markup LanguageпјүжҳҜдёҖдёӘ XML жЎҶжһ¶пјҢд№ҹе°ұжҳҜдёҖз»„еҚҸи®®пјҢеҸҜд»Ҙз”ЁжқҘдј иҫ“е®үе…ЁеЈ°жҳҺгҖӮжҜ”еҰӮпјҢдёӨеҸ°иҝңзЁӢжңәеҷЁд№Ӣй—ҙиҰҒйҖҡи®ҜпјҢдёәдәҶдҝқиҜҒе®үе…ЁпјҢжҲ‘д»¬еҸҜд»ҘйҮҮз”ЁеҠ еҜҶзӯүжҺӘж–ҪпјҢд№ҹеҸҜд»ҘйҮҮз”Ё SAML жқҘдј иҫ“пјҢдј иҫ“зҡ„ж•°жҚ®д»Ҙ XML еҪўејҸпјҢз¬ҰеҗҲ SAML и§„иҢғпјҢиҝҷж ·жҲ‘д»¬е°ұеҸҜд»ҘдёҚиҰҒжұӮдёӨеҸ°жңәеҷЁйҮҮз”Ёд»Җд№Ҳж ·зҡ„зі»з»ҹпјҢеҸӘиҰҒжұӮиғҪзҗҶи§Ј SAML и§„иҢғеҚіеҸҜпјҢжҳҫз„¶жҜ”дј з»ҹзҡ„ж–№ејҸжӣҙеҘҪгҖӮSAML и§„иҢғжҳҜдёҖз»„ Schema е®ҡд№үгҖӮ

еҸҜд»Ҙиҝҷд№ҲиҜҙпјҢеңЁ Web Service йўҶеҹҹпјҢschema е°ұжҳҜи§„иҢғпјҢеңЁ Java йўҶеҹҹпјҢAPI е°ұжҳҜи§„иҢғ

3.11 X509AuthenticationFilter

3.12 AbstractPreAuthenticatedProcessingFilter

дҪ еҸҜд»Ҙз»§жүҝиҜҘзұ»иҝӣиЎҢе…·дҪ“е®һзҺ°е№¶йҖҡиҝҮ HttpSecurity#addFilter ж–№жі•жқҘж·»еҠ дёӘжҖ§еҢ–зҡ„AbstractPreAuthenticatedProcessingFilter гҖӮ

3.25 RequestCacheAwareFilter

Spring Security6 - зҺ©иҪ¬иҮӘе®ҡд№үзҷ»еҪ•

Posted on 2021-07-03 In Spring Security

1. еүҚиЁҖ

е®үе…Ёи®ҝй—®зҡ„з¬¬дёҖжӯҘе°ұжҳҜи®ӨиҜҒпјҲAuthenticationпјүпјҢи®ӨиҜҒзҡ„з¬¬дёҖжӯҘе°ұжҳҜзҷ»еҪ•гҖӮд»ҠеӨ©жҲ‘д»¬иҰҒйҖҡиҝҮеҜ№ Spring Security зҡ„иҮӘе®ҡд№үпјҢжқҘи®ҫи®ЎдёҖдёӘеҸҜжү©еұ•пјҢеҸҜдјёзј©зҡ„ form зҷ»еҪ•еҠҹиғҪгҖӮ

2. form зҷ»еҪ•зҡ„жөҒзЁӢ

дёӢйқўжҳҜformзҷ»еҪ•зҡ„еҹәжң¬жөҒзЁӢ:

еҸӘиҰҒжҳҜ form зҷ»еҪ•еҹәжң¬йғҪиғҪиҪ¬еҢ–дёәдёҠйқўзҡ„жөҒзЁӢгҖӮжҺҘдёӢжқҘжҲ‘д»¬зңӢзңӢ Spring Security жҳҜеҰӮдҪ•еӨ„зҗҶзҡ„гҖӮ

3. Spring Security дёӯзҡ„зҷ»еҪ•

дёҠдёҖзҜҮдёӯе·Із»Ҹи®ІеҲ°дәҶжҲ‘д»¬йҖҡеёёзҡ„иҮӘе®ҡд№үи®ҝй—®жҺ§еҲ¶дё»иҰҒжҳҜйҖҡиҝҮ HttpSecurity жқҘжһ„е»әзҡ„гҖӮй»ҳи®Өе®ғжҸҗдҫӣдәҶдёүз§Қзҷ»еҪ•ж–№ејҸпјҡ

  • formLogin() жҷ®йҖҡиЎЁеҚ•зҷ»еҪ•
  • oauth2Login() еҹәдәҺ OAuth2.0 и®ӨиҜҒ/жҺҲжқғеҚҸи®®
  • openidLogin() еҹәдәҺ OpenID иә«д»Ҫи®ӨиҜҒи§„иҢғ

д»ҘдёҠдёүз§Қж–№ејҸз»ҹз»ҹжҳҜ AbstractAuthenticationFilterConfigurer е®һзҺ°зҡ„гҖӮ

4. HttpSecurity дёӯзҡ„ form иЎЁеҚ•зҷ»еҪ•

еҗҜз”ЁиЎЁеҚ•зҷ»еҪ•йҖҡиҝҮдёӨз§Қж–№ејҸдёҖз§ҚжҳҜйҖҡиҝҮ HttpSecurity зҡ„ apply(C configurer) ж–№жі•иҮӘе·ұжһ„йҖ дёҖдёӘ AbstractAuthenticationFilterConfigurer зҡ„е®һзҺ°пјҢиҝҷз§ҚжҳҜжҜ”иҫғй«ҳзә§зҡ„зҺ©жі•гҖӮ еҸҰдёҖз§ҚжҳҜжҲ‘д»¬еёёи§Ғзҡ„дҪҝз”Ё HttpSecurity зҡ„ formLogin() ж–№жі•жқҘиҮӘе®ҡд№ү FormLoginConfigurer гҖӮжҲ‘д»¬е…ҲжҗһдёҖдёӢжҜ”иҫғеёёи§„зҡ„з¬¬дәҢз§ҚгҖӮ

4.1 FormLoginConfigurer

иҜҘзұ»жҳҜ form иЎЁеҚ•зҷ»еҪ•зҡ„й…ҚзҪ®зұ»гҖӮе®ғжҸҗдҫӣдәҶдёҖдәӣжҲ‘д»¬еёёз”Ёзҡ„й…ҚзҪ®ж–№жі•пјҡ

  • loginPage(String loginPage) : зҷ»еҪ•йЎөйқўиҖҢе№¶дёҚжҳҜжҺҘеҸЈпјҢеҜ№дәҺеүҚеҗҺеҲҶзҰ»жЁЎејҸйңҖиҰҒжҲ‘д»¬иҝӣиЎҢж”№йҖ  й»ҳи®Өдёә /loginгҖӮ
  • loginProcessingUrl(String loginProcessingUrl)е®һйҷ…иЎЁеҚ•еҗ‘еҗҺеҸ°жҸҗдәӨз”ЁжҲ·дҝЎжҒҜзҡ„ ActionпјҢеҶҚз”ұиҝҮж»ӨеҷЁUsernamePasswordAuthenticationFilterжӢҰжҲӘеӨ„зҗҶпјҢиҜҘ Action е…¶е®һдёҚдјҡеӨ„зҗҶд»»дҪ•йҖ»иҫ‘гҖӮ
  • usernameParameter(String usernameParameter) з”ЁжқҘиҮӘе®ҡд№үз”ЁжҲ·еҸӮж•°еҗҚпјҢй»ҳи®Ө usernameгҖӮ
  • passwordParameter(String passwordParameter)з”ЁжқҘиҮӘе®ҡд№үз”ЁжҲ·еҜҶз ҒеҗҚпјҢй»ҳи®Өpassword

1. еүҚиЁҖ

2.2 ж ёеҝғиҝҮж»ӨеҷЁй…ҚзҪ®ж–№жі•

2.3 е®үе…ЁиҝҮж»ӨеҷЁй“ҫй…ҚзҪ®ж–№жі•

exceptionHandling() е…Ғи®ёй…ҚзҪ®й”ҷиҜҜеӨ„зҗҶ
securityContext() еңЁHttpServletRequestsд№Ӣй—ҙзҡ„SecurityContextHolderдёҠи®ҫзҪ®SecurityContextзҡ„з®ЎзҗҶгҖӮ еҪ“дҪҝз”ЁWebSecurityConfigurerAdapterж—¶пјҢиҝҷе°ҶиҮӘеҠЁеә”з”Ё
servletApi() е°ҶHttpServletRequestж–№жі•дёҺеңЁе…¶дёҠжүҫеҲ°зҡ„еҖјйӣҶжҲҗеҲ°SecurityContextдёӯгҖӮ еҪ“дҪҝз”ЁWebSecurityConfigurerAdapterж—¶пјҢиҝҷе°ҶиҮӘеҠЁеә”з”Ё
csrf() ж·»еҠ  CSRF ж”ҜжҢҒпјҢдҪҝз”ЁWebSecurityConfigurerAdapterж—¶пјҢй»ҳи®ӨеҗҜз”Ё
logout() ж·»еҠ йҖҖеҮәзҷ»еҪ•ж”ҜжҢҒгҖӮеҪ“дҪҝз”ЁWebSecurityConfigurerAdapterж—¶пјҢиҝҷе°ҶиҮӘеҠЁеә”з”ЁгҖӮй»ҳи®Өжғ…еҶөжҳҜпјҢи®ҝй—®URLвҖқ/ logoutвҖқпјҢдҪҝHTTP Sessionж— ж•ҲжқҘжё…йҷӨз”ЁжҲ·пјҢжё…йҷӨе·Ій…ҚзҪ®зҡ„д»»дҪ•#rememberMe()иә«д»ҪйӘҢиҜҒпјҢжё…йҷӨSecurityContextHolderпјҢз„¶еҗҺйҮҚе®ҡеҗ‘еҲ°вҖң/login?successвҖқ
anonymous() е…Ғи®ёй…ҚзҪ®еҢҝеҗҚз”ЁжҲ·зҡ„иЎЁзӨәж–№жі•гҖӮ еҪ“дёҺWebSecurityConfigurerAdapterз»“еҗҲдҪҝз”Ёж—¶пјҢиҝҷе°ҶиҮӘеҠЁеә”з”ЁгҖӮ й»ҳи®Өжғ…еҶөдёӢпјҢеҢҝеҗҚз”ЁжҲ·е°ҶдҪҝз”Ёorg.springframework.security.authentication.AnonymousAuthenticationTokenиЎЁзӨәпјҢе№¶еҢ…еҗ«и§’иүІ вҖңROLE_ANONYMOUSвҖқ
formLogin() жҢҮе®ҡж”ҜжҢҒеҹәдәҺиЎЁеҚ•зҡ„иә«д»ҪйӘҢиҜҒгҖӮеҰӮжһңжңӘжҢҮе®ҡFormLoginConfigurer#loginPage(String)пјҢеҲҷе°Ҷз”ҹжҲҗй»ҳи®Өзҷ»еҪ•йЎөйқў
oauth2Login() ж №жҚ®еӨ–йғЁOAuth 2.0жҲ–OpenID Connect 1.0жҸҗдҫӣзЁӢеәҸй…ҚзҪ®иә«д»ҪйӘҢиҜҒ
requiresChannel() й…ҚзҪ®йҖҡйҒ“е®үе…ЁгҖӮдёәдәҶдҪҝиҜҘй…ҚзҪ®жңүз”ЁпјҢеҝ…йЎ»жҸҗдҫӣиҮіе°‘дёҖдёӘеҲ°жүҖйңҖдҝЎйҒ“зҡ„жҳ е°„
httpBasic() й…ҚзҪ® Http Basic йӘҢиҜҒ
addFilterBefore() еңЁжҢҮе®ҡзҡ„Filterзұ»д№ӢеүҚж·»еҠ иҝҮж»ӨеҷЁ
addFilterAt() еңЁжҢҮе®ҡзҡ„Filterзұ»зҡ„дҪҚзҪ®ж·»еҠ иҝҮж»ӨеҷЁ
addFilterAfter() еңЁжҢҮе®ҡзҡ„Filterзұ»зҡ„д№ӢеҗҺж·»еҠ иҝҮж»ӨеҷЁ
and() иҝһжҺҘд»ҘдёҠзӯ–з•Ҙзҡ„иҝһжҺҘеҷЁпјҢз”ЁжқҘз»„еҗҲе®үе…Ёзӯ–з•ҘгҖӮе®һйҷ…дёҠе°ұжҳҜвҖңиҖҢдё”вҖқзҡ„ж„ҸжҖқ
